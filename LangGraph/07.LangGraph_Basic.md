# 2026-04-15 | AI Agent 표준 프로토콜 MCP & LangGraph 기반 RAG 챗봇 구축

본 저장소는 AI 에이전트가 외부 도구를 사용하는 방식을 표준화하는 **MCP(Model Context Protocol)** 기술을 학습하고, 이를 확장하여 **LangGraph와 FastAPI**를 결합한 실무형 **RAG(Retrieval-Augmented Generation) 챗봇 서비스**를 구축한 프로젝트입니다.

---

## 🚀 비즈니스 로직 및 설계 핵심 포인트

### 1. MCP (Model Context Protocol) 도입: "AI의 USB-C"
* **표준화 (Standardization):** 과거에는 서비스마다 API 연동 코드를 개별 구현했지만, MCP는 이를 하나의 프로토콜로 통일합니다.
* **확장성:** 한 번 구축한 MCP 서버(Gmail, Slack 등)는 Cursor, Claude Code, 혹은 커스텀 LangGraph 에이전트 어디서든 즉시 연결 가능합니다.

### 2. LangGraph 기반 ReAct 에이전트
* **추론과 행동 (Reasoning & Acting):** LLM이 단순히 답변하는 것을 넘어, 필요시 `document_search` 도구를 호출하여 지식 베이스를 검색합니다.
* **상태 보존 (Persistence):** PostgreSQL 체크포인터를 사용하여 사용자와의 대화 맥락을 영구적으로 저장하고 관리합니다.

### 3. 실시간 스트리밍 아키텍처 (SSE)
* **사용자 경험 최적화:** `stream_mode="messages"`를 활용하여 AI가 생성하는 토큰을 실시간으로 브라우저에 전달(Server-Sent Events)합니다.

## 🛠 실무형 핵심 코드 예시

### 1. LangGraph 에이전트 구성 (`graph.py`)
이 코드는 **"똑똑한 비서(LLM)에게 서류 가방(RAG Tool)과 수첩(Memory)을 쥐어주는 과정"**입니다.

````python
import os
from langchain_chroma import Chroma
from langchain_openai import ChatOpenAI, OpenAIEmbeddings
from langchain_core.tools.retriever import create_retriever_tool
from langgraph.prebuilt import create_react_agent
from langgraph.checkpoint.postgres.aio import AsyncPostgresSaver

# 1. 벡터 저장소 연결 (RAG를 위한 지식 베이스)
embeddings = OpenAIEmbeddings(model="text-embedding-3-small")
vector_store = Chroma(
    embedding_function=embeddings,
    collection_name="spri_ai_brief",
    persist_directory="./chroma_db",
)
retriever = vector_store.as_retriever(search_kwargs={"k": 3})

# 2. 도구 정의 (에이전트가 사용할 기술)
retriever_tool = create_retriever_tool(
    retriever,
    name="document_search",
    description="AI 산업 동향, 기업 전략, 기술 트렌드 문서를 검색합니다."
)

# 3. 에이전트 초기화 (수첩 역할을 할 DB 연결 포함)
async def init_graph(conn_string):
    llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
    
    # DB에 대화 내용을 저장할 체크포인터 설정
    checkpointer = AsyncPostgresSaver.from_conn_string(conn_string)
    await checkpointer.setup()
    
    # 최종 에이전트 그래프 컴파일
    agent_graph = create_react_agent(
        model=llm,
        tools=[retriever_tool],
        checkpointer=checkpointer,
    )
    return agent_graph

from fastapi import FastAPI

import json

@app.post("/conversations/{chat_id}/chat/stream")
async def chat_stream(chat_id: str, request: ChatRequest):
    config = {"configurable": {"thread_id": chat_id}}
    
    async def event_generator():
        # messages 모드로 토큰 단위 스트리밍 수행
        async for msg, metadata in graph.astream(
            {"messages": [("user", request.message)]},
            config=config,
            stream_mode="messages",
        ):
            # AI가 텍스트 토큰을 생성하는 경우
            if msg.type == "AIMessageChunk" and msg.content:
                yield f"data: {json.dumps({'type': 'token', 'content': msg.content})}\n\n"
            
            # AI가 도구(RAG)를 호출하기 시작하는 경우
            elif msg.type == "AIMessageChunk" and msg.tool_call_chunks:
                for tc in msg.tool_call_chunks:
                    if tc.get("name"):
                        yield f"data: {json.dumps({'type': 'tool_call', 'name': tc['name']})}\n\n"

        yield "data: [DONE]\n\n"

    return StreamingResponse(event_generator(), media_type="text/event-stream")
