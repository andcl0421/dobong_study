##  파이썬 기초 알고리즘: 입출력 및 반복문 제어
### 1. 2438번: 별 찍기 - 1 (반복문의 이해)
####  비즈니스 로직 및 설계 핵심
사용자 입력 처리: int(input())을 통해 출력할 층수를 정수형으로 저장합니다.

반복 범위 설정: range(1, total_layers + 1)를 사용하여 1부터 사용자가 입력한 숫자까지 정교하게 반복 횟수를 제어합니다.

문자열 동적 생성: 파이썬의 문자열 곱셈("*" * n) 특성을 활용하여 별의 개수를 층별로 늘려나갑니다.

#### 🛠 트러블슈팅 (Troubleshooting)
문제: 변수 count를 0으로 초기화한 후 range(1, count + 1)을 사용하여 루프가 한 번도 실행되지 않는 논리적 오류 발생.

원인: range(1, 1)은 시작값과 종료값이 같아져 실제 숫자를 생성하지 않음.

해결: 입력값인 total_layers를 직접 range의 인자로 사용하여 반복 횟수를 동적으로 할당함.

####  완성 코드
Python
# 1. 층수를 입력받습니다.
total_layers = int(input())

# 2. 1층부터 N층까지 차례대로 반복합니다.
for current_layer in range(1, total_layers + 1):
    # 3. 현재 층수(Index)만큼 별을 생성하여 출력합니다.
    print('*' * current_layer)
### 2. 1000번: A+B (표준 입력의 기초)
####  비즈니스 로직 및 설계 핵심
문자열 파싱: split() 함수를 통해 공백으로 구분된 데이터를 리스트로 분리합니다.

일괄 형변환: map(int, ...)를 사용하여 문자열 조각들을 연산이 가능한 정수(Integer) 타입으로 한 번에 변환합니다.

#### 💻 완성 코드
Python
# 공백을 기준으로 두 수를 입력받아 정수로 변환합니다.
first_num, second_num = map(int, input().split())

# 합계를 계산하여 출력합니다.
total_sum = first_num + second_num
print(total_sum)
